This is just a simple grade calculator using the 4.0 scale weighted in the way my school does it, improvements will come in the future.

how to actually use it as a cli (or at least the likely suboptimal way I did it, also this only works on  linux, and probably some other unix-like systems):
1.) compile it, I like gcc, there are no special flags needed for this project
~note that this is where I started doing it in probably a bad way, unfortunately I don't know a better way~
2.) log out of your normal account and into root
3.) navigate to the user file folder your account, copy the binary you generated with your compiler
4.) navigate to the bin folder, paste it into there
5.) log out of root and back into your normal account
